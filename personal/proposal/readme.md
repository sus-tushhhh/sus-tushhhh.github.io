# 💘 Anti Rejection

A playful and interactive mini web project where the "No" button refuses to cooperate 😏  
Hover over it or try to click it… and it runs away!

## 🎯 Concept

This project is a fun take on interactive UI behavior:

- The **"Yes"** button works normally ✅
- The **"No"** button actively avoids being clicked ❌🏃‍♂️

It creates a lighthearted "impossible choice" experience.
